# Full-Stack TypeScript Assignment

A two-part technical assignment demonstrating full-stack TypeScript engineering:

- **[frontend/](./frontend)** — React + TypeScript search interface with reusable hooks (`useLocalStorage`, `useDebounce`) and a polymorphic `Button` component.
- **[backend/](./backend)** — Node.js + Express RBAC (Role-Based Access Control) middleware system with protected routes.

## Structure

```
.
├── frontend/   # React + Vite + TypeScript
└── backend/    # Node.js + Express + TypeScript
```

## Running

Each subproject has its own `README.md` with setup instructions.

```bash
# Frontend
cd frontend && npm install && npm run dev

# Backend
cd backend && npm install && npm run dev
```
